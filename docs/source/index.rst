66-527 Proseminar: Programmierprojekt
=====================================

The module *66-527 Proseminar: Programmierprojekt* of the University of Hamburg serves to 
introduce students into scientific computing and software development on large scale projects.

The module is taught by
    - **Daniela Pfannkuche**
    - **Bodo Krause-Kyora**
    - **Lars-Hendrik Frahm** (lfrahm@physnet.uni-hamburg.de)
